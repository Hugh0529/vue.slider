#vue slider

###Feature
- vue 1.x 轮播器

###api
|  Param | Required | Type | Description |
| :-- | --:| :--: | :-- |
| height | F | Numbber | 高度，默认值为300 |
| isShowNumber | F | Boolean | 是否显示数量 |
| speed | F | Number | slider速度, 单位秒 |
| pics | T | Array | uri数组 |
